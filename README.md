# 🏠 부동산 법률 챗봇

주택임대차보호법, 공인중개사법, 주택법 PDF를 기반으로 부동산 관련 법률 질문에 답변하는 AI 챗봇입니다.

## 🛠 기술 스택

- **Backend**: FastAPI, LlamaIndex, ChromaDB, pypdf
- **Frontend**: React, Vite
- **AI**: OpenAI GPT-4o-mini, text-embedding-3-small
- **모니터링**: LangSmith
- **평가**: RAGAS

## 💡 주요 기능

- 부동산 관련 법률 3개 PDF 통합 질의응답
- 대화 기억 기능 (이전 대화 맥락 유지)
- 스트리밍 답변 (글자씩 실시간 출력)
- 참고 법령 출처 표시
- 법률 전문가 면책 조항
- RAGAS 품질 평가 시각화

## 📊 근로기준법 챗봇 대비 개선 사항

| 항목 | 근로기준법 챗봇 | 부동산 법률 챗봇 |
|------|--------------|----------------|
| 문서 수 | 1개 | 3개 |
| PDF 읽기 | SimpleDirectoryReader | pypdf (직접 구현) |
| 대화 기억 | ❌ | ✅ ChatMemoryBuffer |
| 답변 방식 | 일반 응답 | ✅ 스트리밍 |
| 출처 표시 | ❌ | ✅ 참고 법령 |
| 면책 조항 | ❌ | ✅ |
| 시스템 프롬프트 | ❌ | ✅ |
| RAGAS 시각화 | ❌ | ✅ |

## 🔧 문제 해결 경험

- PDF가 바이너리로 저장되는 문제 → pypdf로 교체
- ChromaDB DLL 보안 차단 → McAfee 제거로 해결
- 질문 표현에 따른 답변 품질 차이 → 시스템 프롬프트 튜닝 및 자주 묻는 질문 법률 용어로 변경

## 📊 RAGAS 품질 평가 결과 (chunk_size 1024 기준)

| 질문 | 충실도 | 관련성 |
|------|--------|--------|
| 전세 계약 기간은 얼마나 되나요? | 1.0 | 1.0 |
| 보증금 돌려받으려면 어떻게 해야 하나요? | 0.7 | 0.9 |
| 임대료 증액 청구 한도는 얼마인가요? | 1.0 | 1.0 |
| **평균** | **0.90** | **0.97** |

## 🔄 추가 개선 사항

### Marker PDF 변환
- 기존 pypdf 방식에서 Marker AI 모델 기반 변환으로 업그레이드
- PDF를 마크다운 형식으로 변환하여 법률 조항 구조 정확하게 인식
- 표, 제목, 조항 번호 등 문서 구조 보존

### 질문 리라이팅
- 사용자의 일상적인 질문을 법률 용어로 자동 변환
- 예시: "전입신고는 언제까지?" → "임차인 주민등록 대항력 요건"
- 법률 용어를 몰라도 정확한 답변 제공

### 스트리밍 답변
- 답변이 한 번에 나오는 방식에서 글자씩 실시간 출력으로 개선
- ChatGPT와 유사한 사용자 경험 제공

### RAGAS 품질 시각화
- 챗봇 답변 품질을 차트로 시각화
- 충실도/관련성 점수를 UI에서 바로 확인 가능
